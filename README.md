# Docker Swarm Graylog observability demo

## System requirements

### CPU

- Minimum: 2 cores
- Recommended: 4 cores

### RAM

- Minimum: 4 Gb
- Recommended: 8 Gb

## Quick start (manual)

1. Git clone this repo
2. Init a swarm
3. Join other nodes to the swarm
4. Set up secrets
5. `docker stack deploy --prune -c docker-compose.yml graylog`
